Al realizar una petición POST a la ruta http://localhost:3000/registro y enviar un objeto JSON con los campos "nombre" y "mensaje", el servidor procesa la información recibida mediante req.body y responde con otro objeto JSON.Respuesta obtenida:{  "estado": "Datos recibidos",  "nombre": "Maria",  "mensaje": "Hola comunidad"}Esto confirma que el servidor recibió correctamente los datos enviados y los devolvió en la respuesta.
